Task Overview & Documentation

This document outlines the process for creating a Web application as part of the Systems Engineer recruitment process at MAQ Software. This is a single-page frontend with documentation additionally for that Task.

Task Details

  • Develop a web application
  • Ensure the site is fully accessible via a public IP (Domain), when deployed.
  • The documentation is attached for the Task.

Frontend Setup Steps

  1. Design a responsive, single-page application using HTML5, CSS3.
  2. Embed the detailed task documentation in the center of the page for easy review.
  3. Deploy the web app to ensure it is publicly accessible.
